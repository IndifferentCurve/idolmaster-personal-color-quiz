Idolmaster image color HEX source workbooks

These XLSX files are kept as local asset references for future data checks or imports.
The running quiz currently uses the embedded data in data.js.

Files:
- million_live_52_image_colors.xlsx: 765PRO ALLSTARS + MILLION LIVE! THEATER image colors.
- shiny_colors_image_colors.xlsx: SHINY COLORS image colors.
- gakuen_idolmaster_image_colors.xlsx: Gakuen Idolmaster image colors.
- cinderella_girls_image_colors.xlsx: Cinderella Girls image colors.
- sidem_image_colors.xlsx: SideM image colors.
- hair_trap_colors.csv: Base hair-color trap values used for every idol; the game applies a subtle random HSL variation at runtime.
