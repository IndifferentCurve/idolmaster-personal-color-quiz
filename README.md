## 한국어

아이돌 일러스트를 보고 정답 이미지 컬러를 고르는 웹 기반 퀴즈 게임입니다.

### 지원 시리즈

- 765PRO ALLSTARS
- 밀리언 스타즈
- 신데렐라 걸즈
- 샤이니 컬러즈
- 학원 아이돌마스터
- SideM

### 주요 기능

- 시리즈별 선택 또는 전체 아이돌 풀 플레이
- Easy, Normal, Hard, Very Hard 난이도
- 직접 입력 또는 프리셋 기반 문항 수 설정
- 다크 모드 / 라이트 모드
- 한국어, 일본어, 영어 UI
- 정답/오답 피드백, 콤보, 오답 노트
- 결과 이미지 저장 및 공유

### 로컬 실행

정적 웹앱이라 별도 빌드 없이 실행할 수 있습니다.

```bash
python -m http.server 8765
```

이후 브라우저에서 아래 주소로 접속합니다.

```text
http://localhost:8765/
```

## English

A web-based quiz game where you look at idol illustrations and choose the correct image color.

### Supported Series

- 765PRO ALLSTARS
- Million Stars
- Cinderella Girls
- Shiny Colors
- Gakuen Idolmaster
- SideM

### Features

- Play by selected series or from the full idol pool
- Easy, Normal, Hard, and Very Hard difficulties
- Question count presets or manual input
- Dark mode / light mode
- Korean, Japanese, and English UI
- Correct/wrong feedback, combo system, and missed-answer notes
- Result image saving and sharing

### Local Run

This is a static web app, so it can run without a build step.

```bash
python -m http.server 8765
```

Then open the following URL in your browser.

```text
http://localhost:8765/
```
