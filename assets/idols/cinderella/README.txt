아이돌마스터 신데렐라 걸즈 공식 포털 스탠딩 이미지 수집본

구성:
- *.webp: 190명 전원의 공식 프로필 스탠딩 이미지를 품질 80 WebP로 변환한 트래픽용 런타임 에셋
- manifest.csv: 아이돌명, 속성, HEX, 파일명, 공식 프로필 URL, 이미지 URL 매핑
- 무손실 WebP 참조본: reference_assets/idols_lossless_webp/cinderella

기준:
- 공식 아이돌 목록: https://cinderellagirls.idolmaster-official.jp/idol/
- 스탠딩 이미지: 각 아이돌 개별 페이지의 IDOL 섹션에 표시되는 기본 전신 스탠딩 이미지
- 정렬: 엑셀과 manifest 모두 속성(큐트 → 쿨 → 패션) 후 색상순
- 파일명: 공식 페이지 슬러그 기준
