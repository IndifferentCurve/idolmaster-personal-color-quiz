Idolmaster personal color HEX source workbooks

These XLSX files are kept as local asset references for future data checks or imports.
The running quiz currently uses the embedded data in data.js.

Files:
- million_live_52_personal_colors.xlsx: 765PRO ALLSTARS + MILLION LIVE! THEATER personal colors.
- shiny_colors_hex.xlsx: SHINY COLORS personal colors.
- gakuen_idolmaster_colors_hex.xlsx: Gakuen Idolmaster personal colors.
- cinderella_girls_personal_colors.xlsx: Cinderella Girls personal colors.
- hair_trap_colors.csv: Base hair-color trap values used for every idol; the game applies a subtle random HSL variation at runtime.
