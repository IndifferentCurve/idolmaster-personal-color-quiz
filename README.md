## 한국어

아이돌 일러스트를 보고 정답 이미지 컬러를 고르는 웹 기반 퀴즈 게임입니다.

### 지원 시리즈

- 765PRO ALLSTARS
- 밀리언 스타즈
- 신데렐라 걸즈
- 샤이니 컬러즈
- 학원 아이돌마스터
- SideM

### 주요 기능

- 시리즈별 출제, 전체 출제, 커스텀 아이돌 직접 선택
- 커스텀 선택 창의 시리즈 필터와 다국어 검색
- Easy, Normal, Hard, Very Hard 난이도
- 문항 수 프리셋, 전체, 직접 입력
- 다크 모드 / 라이트 모드
- 한국어, 일본어, 영어 UI
- 콤보, 정답/오답 피드백, 오답 노트
- 결과 이미지 미리보기, 저장, 공유
- WebP 이미지와 지연 로딩 기반의 가벼운 로딩

### 로컬 실행

정적 웹앱이라 별도 빌드 없이 실행할 수 있습니다.

```bash
python -m http.server 8765
```

이후 브라우저에서 아래 주소로 접속합니다.

```text
http://localhost:8765/
```

- HEX: [imas-db](https://imas-db.jp/misc/color.html)
- 이미지: 각 시리즈 공식/참조 출처 및 `assets` 폴더의 README/manifest 파일 참고
- 신데렐라 걸즈 영문명: [THE iDOLM@STER Cinderella Girls Fandom Characters](https://idolmaster-cinderella-girls.fandom.com/wiki/Characters)

이 프로젝트는 팬메이드 퀴즈입니다. THE IDOLM@STER 및 관련 시리즈의 권리는 각 권리자에게 있습니다.

## English

A web-based quiz game where you look at idol illustrations and choose the correct image color.

### Supported Series

- 765PRO ALLSTARS
- Million Stars
- Cinderella Girls
- Shiny Colors
- Gakuen Idolmaster
- SideM

### Features

- Play by selected series, the full idol pool, or a custom idol selection
- Series filters and multilingual search in custom selection
- Easy, Normal, Hard, and Very Hard difficulties
- Question count presets, all questions, or manual input
- Dark mode / light mode
- Korean, Japanese, and English UI
- Combo system, correct/wrong feedback, and missed-answer notes
- Result image preview, saving, and sharing
- Lightweight loading with WebP assets and lazy-loaded images

### Local Run

This is a static web app, so it can run without a build step.

```bash
python -m http.server 8765
```

Then open the following URL in your browser.

```text
http://localhost:8765/
```
