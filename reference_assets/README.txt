Reference assets preserved for source-quality checks.

The quiz runtime does not load files from this directory.

- idols_lossless_webp: lossless WebP copies of all idol illustrations before runtime compression.
- official_brand_colors.csv: official Idolmaster brand color HEX references from imas-db.
- official_brand_colors.xlsx: the same brand color references with filled color swatches for quick visual checks.

Runtime traffic uses the smaller lossy WebP files in assets/idols.
